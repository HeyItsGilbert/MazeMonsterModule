# New-DiceRoll

## SYNOPSIS
Creates a dice roll.
D6 by default.

## SYNTAX

```
New-DiceRoll [[-Dice] <Dice>] [<CommonParameters>]
```

## DESCRIPTION
Returns a random number

## EXAMPLES

### EXAMPLE 1
```
New-DiceRoll
```

Returns an int which can be used within scripts, or by PC's or GM's.
